API 参考文档

Whois Hacker 提供了 RESTful API，允许您以编程方式查询和处理 WHOIS 信息。本文档详细说明了所有可用的 API 端点、参数和响应格式。

开始使用

要使用 Whois Hacker API，您需要先启动 API 服务：

whois-hacker service --port 8080

默认情况下，API 服务在 http://localhost:8080 上运行。您可以通过 --port 参数指定不同的端口。

身份验证

默认情况下，API 不需要身份验证。如果您需要增加安全性，可以启用 API 密钥：

whois-hacker service --api-key YOUR_SECRET_KEY

启用 API 密钥后，您需要在每个请求中包含 X-API-Key 请求头：

curl -H "X-API-Key: YOUR_SECRET_KEY" "http://localhost:8080/whois?domain=example.com"

速率限制

为了防止滥用，API 服务实施了速率限制。默认情况下，每个 IP 地址每分钟可以发出 60 个请求。您可以在启动服务时自定义这些限制：

whois-hacker service --rate-limit 10 --rate-limit-period 60

上述命令将限制每个 IP 在 60 秒内最多发起 10 次请求。

当您超出速率限制时，API 将返回 HTTP 429 (Too Many Requests) 状态码。响应头 X-RateLimit-Limit、X-RateLimit-Remaining 和 X-RateLimit-Reset 提供了有关限制的详细信息。

错误处理

API 使用标准 HTTP 状态码表示请求状态。常见的状态码包括：

  • 200 OK - 请求成功
  • 400 Bad Request - 请求参数不正确
  • 401 Unauthorized - 缺少或无效的 API 密钥
  • 404 Not Found - 请求的资源不存在
  • 429 Too Many Requests - 超出速率限制
  • 500 Internal Server Error - 服务器内部错误

错误响应的 JSON 格式如下：

{
  "error": true,
  "message": "错误消息",
  "status": 400
}

GET /whois

查询域名的 WHOIS 信息。

查询参数

参数 类型 必需 描述
domain string 是 要查询的域名
server string 否 使用特定的 WHOIS 服务器
format string 否 响应格式（json、text、csv），默认为 json
timeout integer 否 查询超时时间（秒），默认为 10

curl "http://localhost:8080/whois?domain=example.com&format=json"

POST /whois/batch

批量查询多个域名的 WHOIS 信息。

请求体 (JSON)

{
  "domains": ["example.com", "google.com", "microsoft.com"],
  "format": "json",
  "concurrency": 5,
  "timeout": 10
}

请求参数

参数 类型 必需 描述
domains array 是 要查询的域名列表
format string 否 响应格式（json、text、csv），默认为 json
concurrency integer 否 并发查询数量，默认为 3
timeout integer 否 每个查询的超时时间（秒），默认为 10

示例请求

curl -X POST -H "Content-Type: application/json" -d '{"domains": ["example.com", "google.com"]}' "http://localhost:8080/whois/batch"

响应

{
  "results": [
    {
      "domain": "example.com",
      "status": "success",
      "data": {
        // WHOIS 信息，格式与单个查询相同
      }
    },
    {
      "domain": "google.com",
      "status": "success",
      "data": {
        // WHOIS 信息
      }
    }
  ],
  "summary": {
    "total": 2,
    "successful": 2,
    "failed": 0,
    "time_taken": "1.234s"
  }
}

代码示例

Python

import requests

# 查询单个域名
response = requests.get('http://localhost:8080/whois', params={
    'domain': 'example.com',
    'format': 'json'
})
data = response.json()
print(f"Domain: {data['domain']}")
print(f"Registrar: {data['registrar']['name']}")
print(f"Expiration: {data['expiration_date']}")

# 批量查询
domains = ['example.com', 'google.com', 'microsoft.com']
response = requests.post('http://localhost:8080/whois/batch', json={
    'domains': domains,
    'concurrency': 3
})
results = response.json()
for result in results['results']:
    if result['status'] == 'success':
        print(f"Domain: {result['domain']}")
        print(f"Registrar: {result['data']['registrar']['name']}")
    else:
        print(f"Failed to query {result['domain']}: {result['error']}")

JavaScript

// 查询单个域名
fetch('http://localhost:8080/whois?domain=example.com&format=json')
  .then(response => response.json())
  .then(data => {
    console.log(`Domain: ${data.domain}`);
    console.log(`Registrar: ${data.registrar.name}`);
    console.log(`Expiration: ${data.expiration_date}`);
  })
  .catch(error => console.error('Error:', error));

// 批量查询
const domains = ['example.com', 'google.com', 'microsoft.com'];
fetch('http://localhost:8080/whois/batch', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ domains, concurrency: 3 })
})
  .then(response => response.json())
  .then(data => {
    data.results.forEach(result => {
      if (result.status === 'success') {
        console.log(`Domain: ${result.domain}`);
        console.log(`Registrar: ${result.data.registrar.name}`);
      } else {
        console.error(`Failed to query ${result.domain}: ${result.error}`);
      }
    });
  })
  .catch(error => console.error('Error:', error));

Go

作为 Go 开发者，您可以选择直接使用我们的 Go 包，这比调用 HTTP API 更加高效和类型安全：

package main

import (
	"fmt"
	"github.com/cyberspacesec/whois-hacker/pkg/whois"
)

func main() {
	// 单个域名查询
	result, err := whois.ExecuteQuery(&whois.QueryOptions{
		Domain: "example.com",
		ValidateResult: true,
	})
	if err != nil {
		fmt.Printf("查询失败: %v\n", err)
		return
	}
	fmt.Printf("域名: %s\n", result.Domain.Domain)
	fmt.Printf("注册商: %s\n", result.Registrar.Name)
}

使用 Go 包的优势：

  • 类型安全：所有的结构和字段都有明确的类型定义
  • IDE 支持：自动补全和文档提示
  • 性能更好：无需 HTTP 开销
  • 功能更丰富：支持更多高级特性
  • 更好的错误处理：使用 Go 的错误处理机制

高级功能示例：

// 批量查询
aggregator := &whois.QueryAggregator{
    concurrency: 5,
}
domains := []string{"example.com", "google.com", "github.com"}
for _, domain := range domains {
    aggregator.AddQuery(domain, &whois.QueryOptions{
        ValidateResult: true,
        Priority: 1,
    })
}
results := aggregator.ExecuteAll()

// 使用缓存
cache := whois.GetCache()
if entry, exists := cache.Get("example.com"); exists {
    fmt.Printf("从缓存获取: %+v\n", entry)
}

// 设置代理
proxyConfig := &whois.ProxyConfig{
    Enabled: true,
    Type: "socks5",
    Address: "127.0.0.1:1080",
    Username: "user",
    Password: "pass",
    Timeout: 10,
}
if err := whois.SetWhoisProxy(proxyConfig); err != nil {
    fmt.Printf("设置代理失败: %v\n", err)
}

// 查询 ASN 信息
ipv4Ranges, ipv6Ranges, err := whois.GetIPRangesByASN("AS15169")
if err != nil {
    fmt.Printf("查询ASN失败: %v\n", err)
    return
}
fmt.Printf("IPv4范围: %v\n", ipv4Ranges)
fmt.Printf("IPv6范围: %v\n", ipv6Ranges)
