核心功能

多功能、高性能的 WHOIS 查询工具，满足各种专业需求

精准 WHOIS 查询

直接连接到权威 WHOIS 服务器，获取最精确、最新的域名注册信息，支持各类顶级域名。

批量处理

同时查询多个域名，高效并行处理，节约时间。支持从文件导入、输出多种格式结果。

代理支持

通过代理服务器发送请求，避开区域限制，保护隐私，支持 HTTP、SOCKS5 等多种代理类型。

国际化域名支持

完全支持 IDN（国际化域名），包括中文、俄文、阿拉伯文等非 ASCII 字符域名的正确查询和显示。

HTTP API 服务

内置 HTTP API 服务，便于集成到其他系统或应用程序中，提供 RESTful 接口进行远程查询。

MCP AI 集成

通过 Multi-agent Conversation Protocol (MCP) 支持与 AI 助手无缝集成，赋能智能化查询和数据分析。

快速开始

简单几步，立即开始使用 Whois Hacker

1 安装

下载适合您系统的安装包，快速安装：

$ curl -sSL https://install.whois-hacker.example.com | sh
# 或者
$ wget -O whois-hacker.tar.gz https://github.com/cyberspacesec/whois-hacker/releases/latest/download/whois-hacker-linux-amd64.tar.gz
$ tar -xzf whois-hacker.tar.gz
$ sudo mv whois-hacker /usr/local/bin/
查看所有下载选项

2 基本域名查询

查询单个域名信息：

$ whois-hacker example.com

使用代理服务器：

$ whois-hacker --proxy socks5://127.0.0.1:1080 example.com

3 启动 API 服务

启动 HTTP API 服务：

$ whois-hacker --api --port 8080

然后可以通过 API 查询：

$ curl "http://localhost:8080/whois?domain=example.com"
查看所有 API 选项

4 Docker 部署

使用 Docker 快速部署：

$ docker run -p 8080:8080 cyberspacesec/whois-hacker:latest --api --port 8080

或使用 docker-compose：

$ wget -O docker-compose.yml https://github.com/cyberspacesec/whois-hacker/raw/main/docker-compose.yml
$ docker-compose up -d
