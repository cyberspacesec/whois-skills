安装指南

Windows 安装步骤

  1. 下载适合您系统的 Windows 版本（32位或64位）
  2. 解压下载的 ZIP 文件到您选择的目录
  3. （可选）将解压目录添加到系统 PATH 环境变量中，以便在任何目录中使用 whois-hacker 命令
  4. 打开命令提示符或 PowerShell，运行 whois-hacker.exe version 验证安装是否成功
将目录添加到 PATH（可选）
  1. 右键点击"此电脑"或"我的电脑"，选择"属性"
  2. 点击"高级系统设置"
  3. 点击"环境变量"按钮
  4. 在"用户变量"或"系统变量"部分找到 PATH 变量并编辑
  5. 添加包含 whois-hacker.exe 的完整路径
  6. 点击"确定"保存更改

macOS 安装步骤

  1. 下载适合您 Mac 处理器的版本（Intel 或 Apple Silicon）
  2. 打开终端应用程序
  3. 解压下载的文件： 
    tar -xzf whois-hacker-v1.2.0-darwin-amd64.tar.gz
cd whois-hacker-v1.2.0-darwin-amd64
  4. 将可执行文件移动到系统路径中： 
    sudo mv whois-hacker /usr/local/bin/
sudo chmod +x /usr/local/bin/whois-hacker
  5. 验证安装： 
    whois-hacker version
如果您使用 M1/M2 Mac，请下载 arm64 版本，并将文件名中的 amd64 替换为 arm64。

Linux 安装步骤

  1. 下载适合您系统架构的 Linux 版本
  2. 打开终端
  3. 解压下载的文件： 
    tar -xzf whois-hacker-v1.2.0-linux-amd64.tar.gz
cd whois-hacker-v1.2.0-linux-amd64
  4. 将可执行文件移动到系统路径中： 
    sudo mv whois-hacker /usr/local/bin/
sudo chmod +x /usr/local/bin/whois-hacker
  5. 验证安装： 
    whois-hacker version
对于非 x86_64 架构（如 ARM），请下载相应版本并调整上述命令中的文件名。

Docker 安装与使用

  1. 确保已安装 Docker（获取 Docker）
  2. 拉取 Whois Hacker 镜像： 
    docker pull cyberspacesec/whois-hacker:latest
  3. 运行单个域名查询： 
    docker run --rm cyberspacesec/whois-hacker query example.com
  4. 启动 API 服务： 
    docker run -p 8080:8080 --name whois-service cyberspacesec/whois-hacker service
  5. 使用 Docker Compose 部署（创建 docker-compose.yml 文件）： 
    version: '3'

services:
  whois-hacker:
    image: cyberspacesec/whois-hacker
    ports:
      - "8080:8080"
    command: service --port 8080 --rate-limit 100 --rate-limit-period 60
    restart: unless-stopped
  6. 启动 Docker Compose 服务： 
    docker-compose up -d

源码编译安装

  1. 确保已安装 Go 1.16 或更高版本（安装 Go）
  2. 克隆代码库： 
    git clone https://github.com/cyberspacesec/whois-hacker.git
cd whois-hacker
  3. 编译项目： 
    make build
  4. 编译后的二进制文件将位于 bin/ 目录下
  5. （可选）安装到系统路径： 
    sudo cp bin/whois-hacker /usr/local/bin/
sudo chmod +x /usr/local/bin/whois-hacker
  6. 验证安装： 
    whois-hacker version
如果您没有安装 Make，也可以直接使用 Go 命令编译：
go build -o bin/whois-hacker ./cmd/whois-hacker

验证下载

为确保下载的文件未被篡改，您可以验证文件的校验和。所有发布版本都附带 SHA-256 校验和文件。

验证下载完整性
使用 SHA-256 校验和验证
Windows
certutil -hashfile whois-hacker-v1.2.0-windows-amd64.zip SHA256
macOS / Linux
shasum -a 256 whois-hacker-v1.2.0-darwin-amd64.tar.gz

将输出与 checksums.txt 文件中的校验和对比。
