# 单元测试覆盖率 100% 补全实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: `superpowers:subagent-driven-development`
> Steps use checkbox (`- [ ]`) syntax.

**Goal:** 将全部 7 个包(whois / api / mcp / metrics / monitor / security / cmd)的单元测试覆盖率从当前 13.3%~76.6% 提升至 100%,覆盖每一个分支、变量、条件判断;过程中自行跑通全部测试并修复发现的问题。

**Architecture:** 数据流 = 测试输入 → 注入隔离依赖(stub Provider / fake 文件 / httptest / 本地 TCP listener / miniredis) → 被测函数 → 断言。沿用现有三种隔离范式并新增两类:`SetWhoisQueryProvider` 等 stub 接口注入(已用于 query_referral_test.go);`t.TempDir()+os.WriteFile` fake 文件(已用于 storage/bgp/history);`httptest.NewServer`(已用于 rdap_test.go)。新增:`net.Listen` 本地伪 whois/HTTP 代理 server + `WhoisServerManager.UpdateServer(tld, localAddr)` 注入地址,使 `rawWhoisQuery`/`queryDirectContext`/`checkServerHealth`/`Dial` 等硬编码 `:43` 的网络函数离线可达 happy-path;新增 miniredis(纯 Go 内存 Redis)使 `RedisCache`/`RedisStorage` 11 个函数离线达 100%。阻塞循环类(`runServe`/`Start*`)用 `context` 取消或子进程隔离;`os.Exit` 类(`errExit`/`main`)用子进程隔离覆盖。复用现有 `Provider` 接口与全局变量「保存 original + defer 恢复」模式,不重构生产代码。

**Tech Stack:** Go 1.23.2, testify v1 (assert), cobra, logrus, go-redis/v8, gopsutil/v3, likexian/whois-parser;新增依赖 `github.com/alicebob/miniredis/v2`(纯 Go,无外部进程);测试隔离工具 net/httptest/os/t.TempDir。

**Risks:**
- 包级 `sync.Once`/全局变量(`cacheOnce`、`poolOnce`、`libConfigOnce`、`globalStorageProvider`、`globalMetricsOnce`、`asnDetailCache`)跨测试污染 → 缓解:每个测试用「保存 original + defer 恢复」,敏感测试用独立 `TestMain` 或子进程。
- Redis 函数需 miniredis,新增依赖可能影响 CI 缓存 → 缓解:miniredis 是纯 Go 单文件库,`go mod tidy` 后无外部构建依赖。
- `os.Exit`/阻塞 `runServe` 无法在同进程测 → 缓解:`TestMain` + `os/exec` 子进程模式(`if os.Getenv("TEST_SUBPROC")=="1" { ...; os.Exit(...) }`)。
- 硬编码 `whois.radb.net:43`/`whois.iana.org:43` 的解析分支(`queryASNFromRADB`/`GetIPRangesByASN`/`QueryIPWithContext` Step1-3)无 server 注入点 → 缓解:RADB 查询走 `defaultClient.dialer`,可临时替换 `defaultClient` 为指向本地 listener 的客户端;iana 查询同理替换 `defaultClient`。若个别行仍不可达,记录于「已知不可达」附录并尽量用本地 listener 覆盖。
- miniredis 不支持部分 Redis 命令(如 `INFO` 全部 section) → 缓解:`RedisCache.GetStats` 调 `client.Info()` 时对返回字段做容错断言。

```go
// pkg/whois/testutil_test.go
package whois

import (
	"context"
	"io"
	"net"
	"testing"
	"time"

	"github.com/alicebob/miniredis/v2"
)

// fakeWhoisServer 启动一个本地 TCP 伪 whois 服务器:对每个连接回写 response。
// 返回 host:port 形式地址与 cleanup。
func fakeWhoisServer(t *testing.T, response string) (string, func()) {
	t.Helper()
	ln, err := net.Listen("tcp", "127.0.0.1:0")
	if err != nil {
		t.Fatalf("listen: %v", err)
	}
	addr := ln.Addr().String()
	ctx, cancel := context.WithCancel(context.Background())
	go func() {
		for {
			conn, err := ln.Accept()
			if err != nil {
				select {
				case <-ctx.Done():
					return
				default:
					return
				}
			}
			go func(c net.Conn) {
				defer c.Close()
				// 读取客户端请求(whois 查询串),丢弃
				c.SetReadDeadline(time.Now().Add(time.Second))
				io.Copy(io.Discard, c)
				c.Write([]byte(response))
			}(conn)
		}
	}()
	cleanup := func() {
		cancel()
		ln.Close()
	}
	return addr, cleanup
}

// registerLocalWhoisServer 把 tld 映射到本地 addr,返回恢复函数。
func registerLocalWhoisServer(tld, addr string) func() {
	mgr := GetServerManager()
	mgr.mu.Lock()
	orig := mgr.servers[tld]
	mgr.mu.Unlock()
	mgr.UpdateServer(tld, addr)
	return func() { mgr.UpdateServer(tld, orig) }
}

// withStubQueryProvider 临时替换全局 WhoisQueryProvider,返回恢复函数。
func withStubQueryProvider(p WhoisQueryProvider) func() {
	orig := globalWhoisQueryProvider
	globalWhoisQueryProvider = p
	return func() { globalWhoisQueryProvider = orig }
}

// withDefaultClient 临时替换 defaultClient,返回恢复函数。
func withDefaultClient(c *WhoisClient) func() {
	orig := defaultClient
	defaultClient = c
	return func() { defaultClient = orig }
}

// newMiniredis 启动内存 Redis,返回地址与 cleanup。
func newMiniredis(t *testing.T) (string, func()) {
	t.Helper()
	mr, err := miniredis.Run()
	if err != nil {
		t.Fatalf("miniredis: %v", err)
	}
	return mr.Addr(), mr.Close
}
```
